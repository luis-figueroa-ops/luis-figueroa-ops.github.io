Using the Tools
These career tools run on your own Anthropic API key — here's how to get one free.

The JD Analyzer and AI Role Revealer call Claude directly from your browser. No data is stored, no account is required on this site — you just need a free Anthropic API key to power them.

The tools connect directly to Claude's API from your browser. There's no server in the middle — so you authenticate with your own key. It keeps your data private and means you only pay for what you use. For the time being, new Anthropic API accounts automatically receive a $5 free trial credit after initial phone verification.

01
Create a free account at console.anthropic.com
02
Go to API Keys in the left sidebar and click Create Key
03
Copy your key — it starts with sk-ant- — and paste it into any tool below
04
New accounts include free credits. Most analyses cost less than a cent each.
